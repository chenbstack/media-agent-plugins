# MoviePilot 数据导入插件

使用 MoviePilot 管理员用户名和密码登录标准 API，分页读取数据，并调用 Media Agent 宿主提供的通用站点账号、订阅和整理记录能力完成增量导入。不需要在 MoviePilot 侧安装迁移桥或手工创建 Token。

- 插件自己维护连接、分页、增量 hash、暂存、字段转换和结果汇总。
- 宿主只提供权限化的领域能力，不包含 MoviePilot 专用逻辑。
- 迁移范围固定为站点、订阅和整理历史，不需要用户逐项选择。
- 订阅历史固定保存在插件私有数据库中作为快照，不写入宿主全局设置。
- 下载历史和下载文件不会迁移；Downloads 通用 SDK 留给下载类插件使用。
